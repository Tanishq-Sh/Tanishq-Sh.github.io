Notes on Reinforcement Learning

October 28, 2025

I'm currently diving into Reinforcement Learning. One of the foundational concepts is the Bellman equation, which recursively defines the value of a state. For a policy $\pi$, the state-value function $v_\pi(s)$ is:

$$ v_\pi(s) = \sum_{a} \pi(a|s) \sum_{s', r} p(s', r|s, a) [r + \gamma v_\pi(s')] $$

This equation expresses that the value of a state $s$ is the expected return when starting in $s$ and following policy $\pi$. It's a fundamental building block for many RL algorithms.

Here's a simple Python implementation of a value iteration update step using NumPy:


import numpy as np

def value_iteration_update(V, P, R, gamma):
    """
    Performs a single Bellman update.
    
    V: Current value function [S]
    P: Transition probabilities [S, A, S']
    R: Reward function [S, A, S']
    gamma: Discount factor
    """
    S, A, _ = P.shape
    Q = np.zeros((S, A))
    
    for s in range(S):
        for a in range(A):
            # E[R + gamma * V(s')]
            expected_value = np.sum(P[s, a, :] * (R[s, a, :] + gamma * V[:]))
            Q[s, a] = expected_value
            
    new_V = np.max(Q, axis=1)
    return new_V
