Notes on Reinforcement Learning

November 16, 2025

In Monte Carlo Prediction, we define a policy that maps states to actions. We also define the immediate rewards for transitioning between states, but the agent is unaware of this mapping. The agent simply follows the policy from a starting state $s$ (which can be random), with the initial return $V(s)$ set to an arbitrary value. It continues taking actions and moving through states until it reaches the terminal state. Once the episode ends, the agent possesses the history of immediate rewards. Starting from the terminal state, the agent derives the value of each visited state, taking the discount factor into consideration. Each new episode begins with these updated values. As we repeat this process across multiple episodes, the estimated values of all states, $V(s)$, move closer to their true values, $v_{\pi}(s)$.
